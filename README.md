# 定投收益回测工具

一个基于 Streamlit 开发的指数定投策略回测工具，支持自定义定投区间、频率和金额，帮助投资者评估定投策略的历史表现。

## 功能特性

- **多种日期选择方式**
  - 手动选择开始和结束日期
  - 按持有年限自动计算结束日期
  - 随机选择持有时间段（支持固定随机种子）

- **灵活的定投频率**
  - 按日定投
  - 按周定投（可选择周一至周五任意一天）
  - 按月定投（可选择1-28号或月底）

- **基金现实因素模拟**
  - 管理费、托管费（按日计提）
  - 申购费、赎回费
  - 现金比例影响
  - 跟踪误差（固定折扣/随机模拟）

- **全面的回测指标**
  - 总投入本金、期末总资产
  - 累计收益率、年化收益率
  - 平均持仓成本
  - 一次性投入年化收益率（对比参考）

- **投资风险分析**
  - 最大亏损（相对于本金）
  - 最大回撤（从峰值到谷值）
  - 累计亏损时长、亏损时长占比
  - 回本时间、回本日期

- **可视化展示**
  - 资产变动曲线（累计投入 vs 持仓市值）
  - 指数收盘价与持仓均价对比图
  - 累计收益率曲线（含亏损区域标注、回本点标识）

## 安装与运行

### 方式一：使用虚拟环境（推荐）

推荐使用 Python 虚拟环境，避免依赖冲突：

```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
.venv\Scripts\activate

# 安装依赖
pip install streamlit pandas numpy plotly openpyxl

# 运行应用
streamlit run dip_investment_backtest.py
```

### 方式二：直接安装

```bash
pip install streamlit pandas numpy plotly openpyxl
streamlit run dip_investment_backtest.py
```

> **注意**：如果遇到 NumPy 版本兼容问题（如 `AttributeError: _ARRAY_API not found`），请使用虚拟环境方式运行。

## 使用说明

1. 上传数据文件

在左侧边栏点击"选择数据文件"，上传 Excel 格式的历史数据文件。数据文件需包含以下列：
- `日期`：交易日期
- `收盘价`：当日收盘价格

2. 设置回测参数

- 选择定投区间（日期选择方式、开始日期、持有年限等）
- 选择定投频率（按日/按周/按月）
- 设置每次定投金额
- 设置基金现实因素参数（可选）

3. 运行回测

点击"开始回测"按钮，系统将计算并展示回测结果。

## 数据文件格式

Excel 文件应包含以下两列：

| 日期 | 收盘价 |
|------|--------|
| 2020-01-02 | 3500.00 |
| 2020-01-03 | 3520.50 |
| ... | ... |

### 数据转换工具

如果您的数据来自中证指数官网（格式为 `日期Date`、`收盘Close`），可使用数据转换工具：

```bash
.venv\Scripts\python data_converter.py
```

该工具会自动将「原始数据」文件夹下的文件转换为可用格式，并保存到「可用数据」文件夹。

**原始数据格式（中证指数）：**
| 日期Date | 收盘Close |
|----------|-----------|
| 20020104 | 1000.00 |
| 20020105 | 982.79 |

**转换后格式：**
| 日期 | 收盘价 |
|------|--------|
| 2002-01-04 | 1000.00 |
| 2002-01-05 | 982.79 |

示例数据文件：`可用数据/东证红利低波.xlsx`

## 回测逻辑说明

### 定投日期确定

- 如果目标定投日是交易日，则在该日定投
- 如果目标定投日不是交易日，则顺延至下一个交易日定投

### 收益率计算

- **累计收益率** = (期末总资产 - 总投入本金) / 总投入本金 × 100%
- **定投年化收益率** = (1 + 累计收益率)^(1/持有年数) - 1
- **一次性投入年化收益率** = (期末价格 / 期初价格)^(1/持有年数) - 1

### 持仓均价

持仓均价 = 累计投入金额 / 累计持有份额

### 基金现实因素计算

| 因素 | 计算方式 |
|------|----------|
| 申购费 | 每次买入时从投入金额中扣除 |
| 管理费+托管费 | 每日按份额计提扣除 |
| 现金比例 | 股票仓位 = 1 - 现金比例，现金部分无收益 |
| 跟踪误差 | 支持固定折扣或随机模拟两种模式 |
| 赎回费 | 期末从总资产中扣除 |

### 风险指标计算

| 指标 | 计算方式 |
|------|----------|
| 最大亏损 | 累计收益率的最低点 |
| 最大回撤 | 从历史最高点到后续最低点的最大跌幅 |
| 累计亏损时长 | 累计收益率为负的总天数 |
| 回本时间 | 从开始投资到再也不亏损所用的时间 |

## 默认设置

- 日期选择方式：随机持有年限
- 固定随机种子：否
- 定投频率：按周（周一）
- 考虑基金现实因素：是

## 注意事项

1. 本工具仅供学习和研究使用，不构成任何投资建议
2. 历史回测结果不代表未来收益表现
3. 定投策略无法规避市场系统性风险
4. 基金现实因素参数仅供参考，实际费用请以基金合同为准
