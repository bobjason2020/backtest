# 红利低波指数定投回测系统

一个基于 Streamlit 的红利低波指数定投回测分析工具，支持固定定投和智能定投策略，帮助投资者分析和对比不同定投策略的历史表现。

## 功能特性

### 两种分析模式

| 模式 | 说明 |
|------|------|
| **单次回测** | 模拟特定时间段内的定投效果，支持固定定投、智能定投、策略对比 |
| **概率分析** | 分析不同起始时间点的定投收益分布，评估盈利概率和收益区间 |

### 三种定投策略

| 策略 | 说明 |
|------|------|
| **固定定投** | 每次投入固定金额，简单稳健 |
| **智能定投** | 根据策略信号动态调整投入金额，低估值加仓、高估值减仓 |
| **策略对比** | 同时运行固定定投和智能定投，对比两种策略的表现差异 |

### 智能定投策略类型

| 策略类型 | 原理 |
|----------|------|
| **均线偏离** | 根据价格与均线的偏离程度判断估值，偏离越大越低估 |
| **趋势动量** | 根据价格趋势和动量指标判断市场状态 |
| **估值分位** | 根据历史价格分位数判断当前估值水平 |
| **组合策略** | 综合多种策略信号，加权决策 |

### 其他功能

- **示例数据**：内置示例数据，新用户可快速体验
- **基金现实因素**：考虑申购费、管理费、托管费、跟踪误差等实际因素
- **风险评估**：最大回撤、波动率、夏普比率等风险指标
- **可视化图表**：收益曲线、收益分布、风险指标等丰富图表

## 安装与运行

### 环境要求

- Python 3.8+
- pip

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
streamlit run dip_investment_backtest.py
```

## 使用说明

### 1. 数据来源

- **上传数据文件**：上传 Excel 格式的基金净值数据
- **使用示例数据**：使用内置的示例数据快速体验

### 2. 选择分析模式

- **单次回测**：分析特定时间段内的定投效果
- **概率分析**：分析不同起始时间点的收益分布

### 3. 选择定投策略

- **固定定投**：每次投入固定金额
- **智能定投**：根据策略信号动态调整投入金额
- **策略对比**：对比固定定投和智能定投的表现

### 4. 设置参数

- 定投频率：每周、每两周、每月
- 定投金额：每次投入的基础金额
- 时间范围：回测的起止日期

### 5. 智能策略参数（可选）

点击"⚙️ 智能策略参数设置"展开详细参数：
- 阈值设置：极度高估/高估/正常/低估/极度低估的阈值
- 倍数设置：对应估值状态的投入倍数

### 6. 基金现实因素（可选）

点击"⚙️ 基金现实因素"展开详细参数：
- 申购费率、赎回费率
- 管理费率、托管费率
- 现金比例、跟踪误差
