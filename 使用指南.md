# 定投回测工具使用指南

本指南适用于已完成环境部署的用户，帮助您快速上手使用定投回测工具。

## 快速启动

在项目目录下运行以下命令启动应用：

```bash
.venv\Scripts\streamlit run dip_investment_backtest.py
```

启动后，浏览器将自动打开 http://localhost:8501

## 数据准备

### 数据获取

本工具支持任意标的的历史数据回测，只要数据包含「日期」和「收盘价」两列即可，包括但不限于：
- 指数（如沪深300、中证红利等）
- 基金（如ETF、场外基金等）
- 股票

**从中证指数官网获取数据：**

1. 访问 [中证指数浏览器](https://www.csindex.com.cn/#/indices/family/list)
2. 搜索并选择目标指数
3. 选择时间范围
4. 点击「导出行情」下载 Excel 文件

> 下载的文件格式为 `日期Date`、`收盘Close`，需使用数据转换工具转换为可用格式。

### 数据格式要求

数据文件必须是 Excel 格式（.xlsx），包含以下两列：

| 列名 | 说明 |
|------|------|
| `日期` | 交易日期，格式如 2020-01-02 |
| `收盘价` | 当日收盘价格 |

### 使用数据转换工具

如果您的数据来自中证指数官网，需要先进行格式转换：

1. 将原始数据文件放入「原始数据」文件夹
2. 运行转换脚本：

```bash
.venv\Scripts\python data_converter.py
```

3. 转换后的文件将保存在「可用数据」文件夹中

**转换示例：**

| 原始格式（中证指数） | 转换后格式 |
|---|---|
| `日期Date`: 20020104 | `日期`: 2002-01-04 |
| `收盘Close`: 1000.00 | `收盘价`: 1000.00 |

## 分析模式

本工具提供两种分析模式：

| 模式 | 说明 |
|------|------|
| 单次回测 | 对选定的时间段进行一次完整的定投回测分析 |
| 概率分析 | 计算在选定时间段内，所有可能起始日期的定投收益概率分布 |

## 单次回测模式

### 第一步：上传数据文件

1. 在左侧边栏点击「选择数据文件」
2. 选择 Excel 格式的历史数据文件（.xlsx）
3. 数据文件必须包含以下两列：
   - `日期`：交易日期
   - `收盘价`：当日收盘价格

上传成功后，页面会显示数据概览信息（记录数、日期范围）。

### 第二步：设置定投区间

提供三种日期选择方式：

| 方式 | 说明 |
|------|------|
| 手动选择日期 | 自主选择开始和结束日期 |
| 按持有年限 | 选择开始日期后，自动计算结束日期 |
| 随机持有年限（默认） | 系统随机选择一段持有期 |

**随机持有年限设置：**
- 固定随机种子：默认关闭，每次回测随机选择不同时间段
- 开启后可输入种子值，用于复现之前的测试结果

### 第三步：选择定投频率

| 频率 | 选项 |
|------|------|
| 按日 | 每个交易日定投 |
| 按周（默认） | 选择周一至周五某一天定投，默认周一 |
| 按月 | 选择1-28号或月底定投 |
| 一次性投入 | 在开始日期后第一个交易日一次性投入全部金额 |

> 若定投日为非交易日，系统会自动顺延至下一个交易日。

### 第四步：设置定投金额

输入每次定投的金额（100 - 1,000,000 元），默认 1000 元。

### 第五步：设置基金现实因素（可选）

默认开启「考虑基金现实因素」，可模拟真实基金投资中的费用影响：

**费用参数：**
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 管理费率（年化） | 0.50% | 基金公司收取，每日计提 |
| 托管费率（年化） | 0.10% | 托管银行收取，每日计提 |
| 申购费率 | 0.12% | 买入时一次性扣除 |
| 赎回费率 | 0.00% | 卖出时从总资产扣除 |

**跟踪因素：**
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 现金比例 | 5.0% | 基金保留的现金仓位 |
| 跟踪误差模式 | 固定折扣 | 固定折扣或随机模拟 |
| 跟踪误差（年化） | 0.10% | 基金净值与指数的偏差 |

### 第六步：运行回测

点击「开始回测」按钮，系统将计算并展示结果。

### 单次回测结果解读

#### 回测结果总览

| 指标 | 含义 |
|------|------|
| 总投入本金 | 定投期间累计投入的资金总额 |
| 期末总资产 | 扣除费用后的最终资产价值 |
| 累计收益率 | (期末总资产 - 总投入本金) / 总投入本金 |
| 平均持仓成本 | 总投入本金 / 累计份额 |
| 定投年化收益率 | 按持有时间折算的年化收益 |
| 一次性投入年化收益率 | 假设期初一次性投入的年化收益（对比参考） |
| 定投时长 | 实际投资时长 |
| 定投次数 | 实际定投次数 |

#### 投资风险分析

| 指标 | 含义 |
|------|------|
| 最大亏损 | 相对于投入本金的最大亏损百分比 |
| 最大回撤 | 从历史最高点到后续最低点的最大跌幅 |
| 累计亏损时长 | 累计收益率为负的总天数 |
| 亏损时长占比 | 亏损天数占总交易日的比例 |
| 回本时间 | 从开始投资到再也不亏损所用的时间 |
| 回本日期 | 回本的具体日期 |

#### 理想收益 vs 实际收益（开启现实因素时显示）

对比理想状态（指数收益）与实际状态（扣除费用后）的差异：
- 理想/实际期末资产
- 理想/实际累计收益率
- 理想/实际年化收益率

#### 费用明细（开启现实因素时显示）

| 费用 | 说明 |
|------|------|
| 累计申购费 | 所有定投申购费用之和 |
| 累计管理费 | 管理费+托管费总计 |
| 赎回费用 | 期末赎回费用 |
| 总费用 | 以上费用合计 |

#### 图表说明

**资产变动曲线**
- 蓝线：累计投入本金（稳步上升的阶梯线）
- 绿色虚线：理想持仓市值（开启现实因素时显示）
- 红线：实际持仓市值

**指数收盘价与持仓均价**
- 绿线：指数收盘价走势
- 橙色虚线：理想持仓均价
- 紫色线：实际持仓均价（开启现实因素时显示）

**累计收益率曲线**
- 绿色虚线：理想累计收益率（开启现实因素时显示）
- 红线：实际累计收益率
- 红色半透明区域：亏损期间
- 蓝色虚线：回本点位置

## 概率分析模式

概率分析模式用于计算在选定时间段内，任意起始日期开始定投一定时长后的收益概率分布。

### 参数设置

#### 分析时间段
- 分析开始日期：默认为数据最早日期
- 分析结束日期：默认为数据最晚日期

#### 定投时长
提供预设选项：1年、2年、3年、5年、7年、10年，也可自定义输入。

#### 定投频率
与单次回测相同：按日、按周、按月、一次性投入。

#### 定投金额
每次定投的金额，默认 1000 元。

#### 采样设置
| 采样方式 | 说明 |
|------|------|
| 每月采样（默认） | 每月取一个起始日期进行回测，计算速度快 |
| 每周采样 | 每周取一个起始日期进行回测 |
| 每日采样 | 每个交易日都作为起始日期进行回测，计算较慢 |

> 采样越密集，结果越精确，但计算时间越长。

#### 基金现实因素
与单次回测相同。

### 运行分析

点击「开始概率分析」按钮，系统将：
1. 遍历分析时间段内所有可能的起始日期
2. 对每个起始日期执行定投回测
3. 统计所有回测结果的收益分布

进度条会显示计算进度。

### 概率分析结果解读

#### 核心概率指标

| 指标 | 含义 |
|------|------|
| 盈利概率 | 累计收益率 > 0 的概率 |

## 使用技巧

### 单次回测技巧

1. **对比分析**：多次运行不同时间段的回测，对比定投收益与一次性投入收益的差异

2. **随机测试**：使用「随机持有年限」功能，多次测试不同市场环境下的定投表现

3. **频率对比**：分别测试按日、按周、按月定投的效果差异

4. **成本观察**：关注持仓均价与收盘价的关系，理解定投摊薄成本的效果

5. **风险评估**：关注最大亏损、回本时间等风险指标，评估自己的风险承受能力

6. **费用影响**：对比开启/关闭现实因素的结果，了解费用对收益的实际影响

### 概率分析技巧

1. **时长对比**：分别测试不同定投时长（如3年、5年、10年）的收益概率，了解时长对收益稳定性的影响

2. **市场周期**：选择不同的分析时间段，观察不同市场周期下定投收益的概率分布

3. **风险认知**：关注亏损概率和最大亏损，建立合理的收益预期

4. **决策参考**：根据年化收益概率分布，设定合理的收益目标

## 常见问题

**Q: 定投日遇到节假日怎么办？**

A: 系统会自动顺延至下一个交易日执行定投。

**Q: 为什么定投年化收益率与一次性投入不同？**

A: 定投是分批买入，资金占用时间不同，收益率计算方式也不同。定投在下跌市场中往往能获得更好的平均成本。

**Q: 如何复现之前的随机测试结果？**

A: 勾选「固定随机种子」并使用相同的种子值即可复现。

**Q: 最大亏损和最大回撤有什么区别？**

A: 最大亏损是相对于投入本金的亏损，最大回撤是相对于历史最高点的跌幅。例如投入10万，最高涨到12万，最低跌到9万，则最大亏损为-10%，最大回撤为-25%。

**Q: 回本时间显示"未回本"是什么意思？**

A: 表示在整个回测期间，累计收益率从未稳定转正，投资一直处于亏损状态。

**Q: 基金现实因素参数如何设置？**

A: 建议参考实际基金的费率说明。一般指数基金管理费0.5%、托管费0.1%，ETF申购费约0.01%-0.15%。

**Q: 概率分析的采样方式如何选择？**

A: 每月采样适合快速了解收益分布；每日采样结果更精确但计算较慢。建议先用每月采样快速分析，再用每日采样精细验证。

**Q: 概率分析结果中的盈利概率很高，是否意味着一定会盈利？**

A: 不一定。概率分析基于历史数据，历史表现不代表未来收益。高盈利概率表示在历史数据中，大多数情况下定投该时长是盈利的，但仍存在亏损的可能。

## 项目结构

本项目采用模块化设计，代码结构清晰：

```
红利trae_seed2/
├── dip_investment_backtest.py    # 主程序入口
├── data_converter.py             # 数据格式转换工具
├── modules/                       # 功能模块目录
│   ├── __init__.py               # 模块初始化与导出
│   ├── config.py                 # 配置常量（页面配置、默认参数等）
│   ├── data_loader.py            # 数据加载与验证
│   ├── investment.py             # 投资计算核心逻辑
│   ├── fee_calculator.py         # 费用计算（申购费、管理费等）
│   ├── risk_analyzer.py          # 风险指标分析
│   ├── probability_analyzer.py   # 概率分析核心逻辑
│   ├── chart_renderer.py         # Plotly图表渲染
│   └── ui_components.py          # Streamlit UI组件
├── 原始数据/                      # 中证指数原始数据
└── 可用数据/                      # 转换后的可用数据
```

### 模块职责说明

| 模块 | 职责 |
|------|------|
| config.py | 集中管理所有配置项和常量 |
| data_loader.py | 处理数据文件的加载和验证 |
| investment.py | 定投日期计算、回测计算、日资产计算 |
| fee_calculator.py | 申购费、管理费、赎回费、跟踪误差计算 |
| risk_analyzer.py | 最大亏损、最大回撤、回本时间等风险指标 |
| probability_analyzer.py | 概率分析、收益分布统计 |
| chart_renderer.py | 资产曲线、价格曲线、收益率曲线、概率分布图表 |
| ui_components.py | 侧边栏参数设置、结果展示等UI组件 |
