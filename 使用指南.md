## 快速启动

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
streamlit run dip_investment_backtest.py
```

### 3. 使用示例数据

1. 在左侧边栏选择"使用示例数据"
2. 选择分析模式和定投策略
3. 点击"开始回测"或"开始概率分析"

## 分析模式

系统提供两种分析模式：

| 模式 | 用途 | 适用场景 |
|------|------|----------|
| **单次回测** | 模拟特定时间段内的定投效果 | 验证特定时间段的投资表现 |
| **概率分析** | 分析不同起始时间点的收益分布 | 评估长期投资的盈利概率 |

---

## 单次回测模式

### 基本设置

1. **选择分析模式**：单次回测
2. **选择定投策略**：固定定投 / 智能定投 / 策略对比
3. **设置定投频率**：每周 / 每两周 / 每月
4. **设置定投金额**：每次投入的基础金额
5. **选择时间范围**：回测的起止日期
6. **点击"开始回测"**

### 定投策略说明

| 策略 | 说明 | 适用人群 |
|------|------|----------|
| **固定定投** | 每次投入固定金额 | 追求简单稳健的投资者 |
| **智能定投** | 根据策略信号动态调整投入金额 | 希望优化投入时机的投资者 |
| **策略对比** | 同时运行固定定投和智能定投 | 想对比两种策略差异的投资者 |

### 回测结果解读

#### 核心指标

| 指标 | 说明 |
|------|------|
| 累计收益率 | 投资期间的总收益率 |
| 年化收益率 | 换算为年度的收益率 |
| 最大回撤 | 期间最大亏损幅度 |
| 波动率 | 收益的波动程度 |
| 夏普比率 | 风险调整后的收益指标 |

#### 图表说明

- **资产曲线图**：展示资产随时间的变化
- **回撤曲线图**：展示期间的回撤情况
- **投资记录表**：每次定投的详细信息

## 概率分析模式

### 基本设置

1. **选择分析模式**：概率分析
2. **选择定投策略**：固定定投 / 智能定投 / 策略对比
3. **设置定投频率**：每周 / 每两周 / 每月
4. **设置定投金额**：每次投入的基础金额
5. **设置分析参数**：
   - 分析时间范围
   - 定投时长（年）
   - 采样方式
6. **点击"开始概率分析"**

### 定投策略说明

| 策略 | 说明 |
|------|------|
| **固定定投** | 分析固定定投在不同起始时间点的收益分布 |
| **智能定投** | 分析智能定投在不同起始时间点的收益分布 |
| **策略对比** | 同时分析两种策略，对比智能定投的胜率 |

### 概率分析结果解读

#### 核心指标

| 指标 | 说明 |
|------|------|
| 盈利概率 | 收益率为正的比例 |
| 平均收益 | 所有模拟的平均收益率 |
| 中位数收益 | 收益率的中位数 |
| 平均年化 | 换算为年度的平均收益率 |

#### 图表说明

- **收益分布图**：展示收益率的分布情况
- **累计概率图**：展示不同收益水平的概率
- **时间线图**：展示收益随起始日期的变化

### 策略对比概率分析

选择"策略对比"时，额外展示：

| 指标 | 说明 |
|------|------|
| 智能策略胜率 | 智能定投收益超过固定定投的比例 |
| 平均收益差异 | 智能定投与固定定投的平均收益差 |
| 收益分布对比图 | 两种策略的收益分布叠加对比 |

## 智能定投策略

### 策略类型

| 策略 | 原理 | 适用场景 |
|------|------|----------|
| **均线偏离** | 价格低于均线越多，越低估 | 震荡市场 |
| **趋势动量** | 根据趋势和动量判断市场状态 | 趋势市场 |
| **估值分位** | 根据历史价格分位数判断估值 | 长期投资 |
| **组合策略** | 综合多种策略信号 | 综合场景 |

### 参数设置

点击"⚙️ 智能策略参数设置"展开详细参数：

#### 估值阈值设置

| 估值状态 | 说明 | 默认阈值（均线偏离） |
|----------|------|---------------------|
| 极度高估 | 市场极度高估，大幅减仓 | +10% |
| 高估 | 市场高估，适度减仓 | +5% |
| 正常 | 市场正常，正常投入 | - |
| 低估 | 市场低估，适度加仓 | -5% |
| 极度低估 | 市场极度低估，大幅加仓 | -10% |

#### 投入倍数设置

| 估值状态 | 默认倍数 | 说明 |
|----------|----------|------|
| 极度高估 | 0.5x | 减半投入 |
| 高估 | 0.75x | 减少投入 |
| 正常 | 1.0x | 正常投入 |
| 低估 | 1.5x | 增加投入 |
| 极度低估 | 2.0x | 双倍投入 |

### 策略信号示例

以均线偏离策略为例：

```
当前价格：1.000
均线价格：1.050
偏离程度：(1.000 - 1.050) / 1.050 = -4.76%

判断：-5% < -4.76% < 0，属于"正常"状态
投入倍数：1.0x（正常投入）
```

## 基金现实因素

### 功能说明

基金投资存在多种费用和跟踪误差，开启此功能可以更真实地模拟实际投资效果。

### 参数设置

点击"⚙️ 基金现实因素"展开详细参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 申购费率 | 买入时的手续费率 | 0.10% |
| 赎回费率 | 卖出时的手续费率 | 0.00% |
| 管理费率 | 年化管理费率 | 0.50% |
| 托管费率 | 年化托管费率 | 0.10% |
| 现金比例 | 基金保留的现金比例 | 5% |
| 跟踪误差 | 基金与指数的跟踪误差 | 0.10% |

### 费用影响

| 费用类型 | 影响方式 |
|----------|----------|
| 申购费 | 每次买入时扣除，减少实际买入份额 |
| 管理费/托管费 | 按日计提，降低基金净值增长 |
| 现金比例 | 部分资金不参与投资，降低收益 |
| 跟踪误差 | 基金与指数的偏离，可能增加或减少收益 |

## 常见问题

### Q1: 为什么我的回测结果与实际情况有差异？

A: 可能的原因：
1. 数据精度问题
2. 未开启"基金现实因素"
3. 定投日期与实际交易日不一致
4. 费率设置与实际不符

### Q2: 智能定投一定比固定定投好吗？

A: 不一定。智能定投在震荡市场表现较好，但在单边上涨市场可能跑输固定定投。建议使用"策略对比"功能分析不同市场环境下的表现。

### Q3: 如何选择合适的定投频率？

A: 
- **每周**：适合资金充裕、希望平滑波动的投资者
- **每月**：适合工薪阶层，与工资发放周期匹配
- **每两周**：介于两者之间

### Q4: 概率分析的采样方式如何选择？

A: 
- **每月采样**：计算速度快，结果代表性好（推荐）
- **每周采样**：结果更精细，计算时间更长
- **每日采样**：结果最精细，计算时间最长

### Q5: 如何解读夏普比率？

A: 
- **< 1**：风险调整后收益较差
- **1-2**：风险调整后收益良好
- **> 2**：风险调整后收益优秀

### Q6: 概率分析为什么这么快？

A: v1.3.9 版本引入了并行计算功能：
- 使用多线程并行计算（ThreadPoolExecutor）
- 自动根据 CPU 核心数选择最优线程数
- 相比串行计算，速度提升 2-4 倍
- 分析完成后会显示运行时间

### Q7: 如何理解运行时间？

A: 运行时间显示在进度条完成后，例如：
- "分析完成！共 120 次模拟，耗时 3.5 秒"
- 运行时间受 CPU 性能、数据量、采样方式等因素影响
- 相同参数的重复计算会利用缓存，速度更快
